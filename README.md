## Platform Support

### Linux
Linux users can use the GUI application or the AppImage release.

### macOS
macOS users can use the command-line interface (CLI).

At this time, there is **no native macOS GUI build** included.  
macOS support is currently provided through the CLI workflow.

---

## macOS CLI Usage

This project includes a no-GUI CLI workflow for macOS users.

### Requirements

- Python 3
- Pillow
- pyserial

Install dependencies:

```bash
pip3 install Pillow pyserial
